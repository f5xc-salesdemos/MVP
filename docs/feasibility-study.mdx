- Surveyed the commercial and open-source agent landscape; found mature tooling optimized for software development, not for augmenting subject matter experts.
- Confirmed that an F5 Distributed Cloud-specialized agent is viable by building on an extensible open-source foundation rather than customizing a general-purpose product.
- Established a make-vs-borrow position: borrow the agent runtime from open source, build the F5 Distributed Cloud domain layer ourselves.
